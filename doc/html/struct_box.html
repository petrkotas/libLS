Detailed Description

template<typename type, int dim>
struct Box< type, dim >

Class representing simple bounding box.

Bounding box has one static method used for detecting intersection between two of them.

Template Parameters
typeSpecifies the type of bounding box (double, int, ...)
dimSpecifies the dimension of the bounding box (2 or 3)
